# 📊 Sales Dashboard

## Project Objective
Analyze sales performance using Power BI and provide interactive insights for business decision-making.

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## Dashboard Features
- Monthly Sales Trend
- Category-wise Sales
- Region-wise Sales
- Top Products
- Customer Analysis
- Interactive Filters

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

🚧 In Progress
